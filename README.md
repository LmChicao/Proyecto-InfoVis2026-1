# Proyecto-InfoVis2026-1 
## **Estudiantes:** Ninoska Cid, Lucas Chicao e Isidora Gomez

Proyecto que trata sobre la visualizacion de la contaminacion del aire a nivel global, 
aqui se identifican las principales causan junto a los paises que estan calificados con la peor calidad de aire

### **Datos:**
Tenemos un dataset con mas de 23 mil datos donde se registran los nombres de paises, ciudades y como los afecta los 4 principales contaminanes medidos, los cuales son Nitrogen Dioxide [NO2], Ozone [O3], Carbon Monoxide [CO] y Particulate Matter [PM2.5].
